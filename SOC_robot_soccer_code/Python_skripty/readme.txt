Praktická část práce pro soutěž Středoškolská odborná činnost.
Práce je na  téma analýzy dat s využitím neuronových sítí. Data jsou získáne ze simulátoru Fotbalu robotů, který poskytla VŠB - Technická univerzita Ostrava.

Skripty zabývající se tvorbou neuronových sítí slouží k tvorbě neuronovývh sítí na základě rozličných vstupních dat. Tyto neuronové sítě 
jsou následně nahrány do simulátoru, který nám poskytuje výstupní data.
Tyto data jsou následně zpracovány ostatními skripty a výsledkem jsou teplotní mapy, které zobrazují chování sítě v simulátoru.
Na základě toho poté můžeme rozhodnonout o úspěšnosti dané sítě
